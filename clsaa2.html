第二节，鼠标的onmouseout，onmousemove事件改变div的样式

函数解释:
使用 onmouseover="toGreen()" onmouseout="toRed()"，调用toGreen()以及toRed()函数，
这里个函数 function toGreen() { var oDiv1 = document.getElementById('div1'); oDiv1.style.width = '200px;'; oDiv1.style.height = '200px'; oDiv1.style.background = 'green'; } function toRed() { var oDiv1 = document.getElementById('div1'); oDiv1.style.width = '100px'; oDiv1.style.height = '100px'; oDiv1.style.background = 'red'; }
用于改变div的颜色以及大小 注意：id的样式的优先级一定大于clsaa的优先级
